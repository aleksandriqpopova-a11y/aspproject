# Finance Expense Tracker

## Overview

**Finance Expense Tracker** е уеб приложение, разработено с ASP.NET Core MVC, което позволява на потребителите да създават и управляват личните си разходи.  
Приложението дава възможност за добавяне на разходи, организиране по категории и проследяване на направените плащания.

Целта на проекта е да демонстрира работа с ASP.NET Core MVC архитектура, Entity Framework Core и работа с база данни, както и реализация на основни CRUD операции.

---

# Project Goals

Основната цел на проекта е да се създаде проста система за управление на разходи, която позволява:

- добавяне на нови разходи
- съхраняване на данни в база
- валидиране на входните данни
- преглед на всички разходи
- използване на MVC архитектура

# Application Architecture

Проектът е структуриран според MVC архитектурата.

### Model
Моделите съдържат данните и правилата за валидация.

### View
Views са HTML страници, които показват информацията на потребителя.

### Controller
Контролерите обработват заявките от потребителя и комуникират с моделите и изгледите.
